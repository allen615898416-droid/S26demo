# 三星 S26（海外版）核心 AI 特性与演示

## 📱 项目简介

这是一个展示三星 Galaxy S26 海外版核心 AI 特性的演示页面，包含 4 大核心功能的实机演示视频。

## ✨ 核心特性

### 1. 🤖 Agentic AI 跨应用调度
系统级智能体，通过一句话指令自动完成多应用协同操作

### 2. 🔍 Ask AI 深度检索
集成 Perplexity 引擎，对多个网页进行交叉比对和结构化摘要

### 3. 🎯 多目标检索
升级版画圈搜索，一次性识别并检索多个视觉目标

### 4. 🛡️ 端侧通话过滤与物理防窥屏
本地 AI 模型实时反诈拦截 + 硬件级隐私防护

## 🚀 在线预览

直接打开 `index.html` 即可在浏览器中查看完整演示。
